# My Personal Portfolio 🚀 (Task 1)

Welcome! This is the repository for my personal portfolio website, built for Task 1 of my Full Stack Web Development internship at Future Interns. 

The goal of this project is to create a clean, modern digital home that showcases what I can build, the technologies I use, and my journey as a developer.

## 🛠️ What I Used to Build It
* **Frontend:** React (powered by Vite for speed)
* **Styling:** Tailwind CSS (for a clean, responsive layout)
* **Hosting:** Vercel / GitHub Pages

## ✨ Core Features
* **About Me & Timeline:** A quick look into my background, my education, and my coding journey.
* **My Tech Stack:** A breakdown of the languages and tools I work with (including Java, Python, JavaScript, Node.js, and databases like MySQL and MongoDB).
* **Project Showcase:** A grid layout displaying my favorite coding projects, complete with tech tags and quick links to the code.
* **Contact Form:** A simple, validated form where anyone can drop me a message.
* **Mobile Friendly:** Designed to look great whether you're looking at it on a phone, tablet, or desktop.
